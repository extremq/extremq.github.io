Let's look at a rather obscure but interestingly useful property of integers.

By asking what the last digit of \(n\) actually means trying to compute \(n \text{ mod } 10\).
Before I show you the formula, I need to talk a bit about .

Euler's totient function is denoted with \(\varphi(n)\) and it counts the positive integers up to \(n\) that are relatively prime to \(n\).
For example, $$ \begin{equation} \varphi(9) = 6 \end{equation} $$ because $$ \begin{equation} 1, 2, 4, 5, 7 \text{ and } 8 \end{equation} $$ do not share any common divisor with \(9\) except for \(1\).

Or, in mathematical terms, \(\text{gcd}(a, 9) = 1\), where \(a \in \{1, 2, 4, 5, 7, 8\}\).

Other examples are \(\varphi(13) = 12\) and \(\varphi(12) = 4\), just so you get the hang of it – and yes, the totient function of a prime number \(p\) returns \(p-1\).

Now that you know what this function is for, let's check out the formula that will be used to aid with our hardships.
Drum roll, please. $$ \begin{equation} a^b \equiv a^{b \text{ mod } \varphi(c)} \text { mod } c \end{equation} $$ If it looks complicated, don't worry. What this notation actually implies is the following:
$$ \begin{equation} a^b \text{ mod } c = a^{b \text{ mod } \varphi(c)} \text { mod } c \end{equation} $$ I'll just show an example.
We know that \(3^7 = 2187\). That means its last digit is \(7\), so \(3^7 \text{ mod } 10 = 7\).
However, using our formula, we can write it as such: $$ \begin{equation} 3^7 \text{ mod } 10 = 3^{7 \text{ mod } \varphi(10)} \text { mod } 10 \end{equation} $$ and that is equivalent to $$ \begin{align} 2187 \text{ mod } 10 &= 3^{7 \text{ mod } 4} \text { mod } 10 \\ &= 3^3 \text { mod } 10 \\ &= 27 \text { mod } 10 \\ &= 7 \end{align} $$ That is true.

I sure do. Let's compute the last digit of \(n=123456789^{123456789}\), once and for all.

To begin with, \(\varphi(10) = 4\). Using this, \(123456789 \text{ mod } \varphi(10)\) is simply \(123456789 \text{ mod } 4 = 1\).
Now, it's trivial – we computed the modified exponent and it resulted to it being \(1\).
That means: $$ \begin{align} 123456789^{123456789} \text{ mod } 10 &= 123456789^1 \text { mod } 10 \\ &= 123456789 \text{ mod } 10 \\ &= 9 \end{align} $$ Who would've guessed?!

Greetings!

You're probably reading this article because you want to be safe online.
I'll tell you what steps you should take whenever you come across a link and what to know beforehand.

Why should you know this?


Fake links are one of the most popular ways of scamming people online.
People tell you that you've won a prize, they give you a link and you must log in with some account of yours.

But, guess what: the site was a replica and it stole your email and password. You probably use the same password for several accounts, so now they will try stealing as many accounts as they can.

I don't think you want this to happen, but sometimes you place your trust in the wrong person/company.
It's extremely easy to avoid these kinds of scams. Let's see how:

0. Clicking on a link isn't dangerous


That's right, it's rule 0, an intro.

If you click on a link and it takes you to a shady website, you can safely close it without any worries.
However, you should not interact at all with it. That means typing personal info (passwords, emails), running downloaded files (simply delete them), allowing access to microphone, webcam or notifications, etc.

That's the way they scam you.

Browsers (such as Chrome, Firefox, Safari, Opera) are your friends: make sure you check the top part of your browser each time you click on a link. Secure site, has a lock. Unsecure site, lock missing.

1. Always read the link before clicking


Why is this so important? Take these three links, for example:
https://instagarm.com/
http://instagram.19exta123jae.com/
https://instagram.io/
When you click them, all show you the Instagram login, exactly as you expect it.

Now, my question is:
Which of these links is fake?

The correct answer:
All of them. The real site is https://instagram.com/.

If this wasn't your answer, you could be a potential victim. Let's see why these links are fake.

2. Check for mistakes or similar-looking names


The first link I showed you was https://instagarm.com/. Read it, letter by letter.

It's intentionally written wrong to look like Instagram, but instead it's Instagarm.
This happens a lot:
https://faecbook.com, https://gmeil.com, https://twittr.com, https://disc0rd.com, etc.

Always, always make sure the name is written correctly.

3. Correctly identify the website name.


Website links can take a lot of forms. For example, the site https://facebook.com may use the following:
https://app.facebook.com,
https://discover.games.facebook.com,
https://groups.facebook.com/post/123.

They all are part of https://facebook.com.
It may be confusing at first, but there's actually a stupidly simple rule:

Only the last two words separated by a dot '.', before the first slash '/', if it exists, tell you the website name.
('https://' and 'http://' do not matter)

If you check that with the examples above, you should get facebook.com.

If it's hard to remember this, the browser also helps you with this: The correct website domain stands out with a striking white while the unimportant rest is gray.
If you remember, I tested you with this link: http://instagram.19exta123jae.com/.
Using the rule, you should confirm that this site is actually 19exta123jae.com, not Instagram. The browser also supports this: The striking white part is 19exta123jae.com.

4. Check the beginning of the link


Sometimes, links start with http:// or https://.

What's the difference? Well, they are the same, except for an extra 's'. It's http and an 's'.
The extra 's' stands for secure and it means everything you give to the website is protected from other people.

Why does this matter? Submitting your password, credit card info or any personal info on a non-secure website is a major threat to your privacy.

Avoid sites that start with http and miss the 's' if they ask you for anything personal.

Again, browsers tell you when such threat exists:
Secure sites have a lock, While unsecure sites do not:

5. Google the website name


Sometimes, websites pass all the rules but you still can't trust them.

In the last example I gave when I tested you, I asked about https://instagram.io/.
Why is this website fake?

The '.io' part does not match with '.com'.

I can, for example, create a website called instagram.gg.
People would not suspect that it's a fake, but the truth is I have no relation with the actual Instagram, so I can fool people into thinking they can trust my website.

This is why you should Google the whole name. Sometimes, they can help you catch the fakes.

Conclusion


The rules are:

1. Always read the link before clicking
2. Check for mistakes or similar-looking names
3. Correctly identify the website name.
4. Check the beginning of the link
5. Google the website name

In addition:

1. Don't be afraid to open a link, but only give information if you can make sure you can trust it.
2. Never ever download a file and open it without FULLY trusting the source.
3. Lastly, don't forget to have fun exploring. Don't be scared to explore the web, but never trust anything without making sure.

Hope it helped you learn something new. Be safe, reader.
